### 🚀 部署就绪

项目已完全准备好用于生产环境：

1. **安装方式**
   ```bash
   # 从源码安装
   pip install -e .
   
   # 或直接使用
   python3 -c "from media_parser_sdk import MediaParser; print('SDK可用')"
   ```

   ```python

   media_info = parse_url("https://www.xiaohongshu.com/explore/...")
   ```

